AutoRamp provides a seamless way for businesses to integrate fiat-to-crypto (onramp), crypto-to-fiat (offramp), and token swap functionality into their applications.

Our API is designed to be simple, secure, and reliable, allowing you to focus on building your core product while we handle the complexities of the crypto-fiat bridge.

## Key Features

- **Onramp**: Enable your users to buy CNGN with local bank transfers.
- **Offramp**: Allow users to sell CNGN/USDC and receive NGN directly in their bank accounts.
- **Swaps**: Swap between USDC and CNGN at competitive market rates.
- **Webhooks**: Get real-time notifications for transaction status changes.

## Getting Started

To start using our API, you'll need an API key, which can be generated from the admin portal by your account manager.
